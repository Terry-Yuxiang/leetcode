### 树类问题
[1245. Tree Diameter](https://leetcode.com/problems/tree-diameter/)   
简化一下题目要求，就是求图中两个node的最远距离是多少。  
思路：  
1.先找一个点，以这个点为root，找到最远的点。  

2.再以最远的点为root，找到距离这个点最远的点。   

3.找到的距离就是图中的最远距离。
